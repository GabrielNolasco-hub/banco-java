## Como Rodar o

1. Abra o terminal na pasta onde estão os arquivos `.java`.
2. Compile os arquivos com o comando:


javac *.java


3. Depois execute o programa com:


java Main


O sistema abrirá um menu no terminal para realizar as operações do banco.

---

## Funcionalidades

O sistema permite realizar algumas operações básicas de um banco:

* Criar conta (corrente ou poupança)
* Listar contas cadastradas
* Depositar dinheiro em uma conta
* Sacar dinheiro
* Transferir dinheiro entre contas
* Consultar saldo de uma conta
